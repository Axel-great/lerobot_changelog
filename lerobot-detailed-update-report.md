## v0.5.0 (2026-03-09) 🔥 最新版本

### 为什么更新
这是 LeRobot 的里程碑版本，带来了大量核心改进和依赖升级。

### 更新内容

#### 🔧 依赖和兼容性
- **Python 3.12+ 最低版本要求** - 提升语言特性和性能
- **transformers v5** - 升级到最新版本
- **opencv-python-headless 上限调整** - 兼容性修复
- Intel RealSense SDK 同步更新

#### 🤖 新增机器人
- **Unitree G1 WBC (Whole Body Control)** - 完整的身体控制实现

#### 📊 训练改进
- **cudnn_deterministic 选项** - 支持可复现训练

#### 📝 其他改进
- 修复 rerun "frame_index" 时间线与 behavior1k 数据集兼容性问题
- 数据集编辑工具改进 - 明确 `root` 参数用法
- 修复 Parquet 文件写入时的元数据索引问题
- 数据集转换脚本修复冗余帧计数
- Slurm 计算资源脚本
- 文档改进：添加 uv 安装说明
- 更新引用 - ICLR26 论文

## v0.4.4 (2026-02-27)

### 为什么更新
这是 v0.4.x 系列的最终版本，带来多个新机器人支持和 API 改进。

### 更新内容

#### 🤖 新增机器人
- **OpenArm** - 新型开源机械臂，支持双手机器人配置
- **Unitree G1** - 遥操作支持

#### 🔌 电机和驱动
- **Damiao 电机 & CAN bus** - 新增电机支持

#### 📷 相机 API
- **相机 API 大规模扩展** - 改进相机后端配置
- ZMQ 相机方法添加清晰的未实现警告

#### 📊 数据集工具
- **subtask 支持** - 数据集支持子任务
- **数据集聚合修复** - 修复子数据集已合并过的问题

#### 🧠 策略改进
- **X-VLA** - 改进训练策略
- **SmolVLA** - wandb 警告修复
- SAC 温度属性修复 - 解决检查点恢复 bug

#### 🌐 环境
- **LIBERO** - 添加任务 ID 配置
- LIBERO init_state_id 重置时变更

## 总结
